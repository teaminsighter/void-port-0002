# Bali Luxe Estates — Luxury Real Estate Landing Page

A production-ready, single-page luxury real estate website with GSAP scroll animations, property showcase, cookie consent, analytics integration, and full SEO.

## Quick Start

1. Edit `config.js` — change company name, phone, email, social links, analytics IDs, form endpoint
2. Update domain in `robots.txt` and `sitemap.xml`
3. Replace Unsplash image URLs with real property photos
4. Deploy to Netlify/Vercel

## Files

| File | Purpose |
|------|---------|
| `index.html` | Main page with all sections |
| `style.css` | Complete styles + responsive + accessibility |
| `script.js` | GSAP animations + form validation + event tracking |
| `config.js` | **Client config** — company info, analytics, form endpoint |
| `robots.txt` | Search engine crawl rules |
| `sitemap.xml` | SEO sitemap |
| `404.html` | Styled error page |

## Features

- GSAP ScrollTrigger animations with property showcase slider
- Cookie consent banner (GDPR compliant — analytics blocked until consent)
- GTM / GA4 / Facebook Pixel support
- Form validation with Formspree/Getform integration
- Full SEO (OG tags, Twitter cards, JSON-LD structured data)
- Accessibility (skip link, sr-only labels, focus-visible, reduced-motion)
- Lazy loading on all below-fold images
- Responsive design (desktop, tablet, mobile)
- Custom cursor, parallax effects, sunset footer animation
