# DONUTS — Handcrafted Bakery Landing Page

An interactive, scroll-driven bakery landing page featuring a 3D donut model (Three.js + GLTFLoader), GSAP ScrollTrigger animations, and Lenis smooth scrolling.

## Quick Start

1. Open `index.html` in a browser (requires ES module support)
2. Customize text, colors, and branding in `index.html` and `style.css`
3. Deploy to Netlify/Vercel

## Files

| File | Purpose |
|------|---------|
| `index.html` | Main page with all sections |
| `style.css` | Complete styles + responsive breakpoints |
| `script.js` | Three.js 3D donut + GSAP scroll animations + Lenis smooth scroll |

## Features

- 3D donut model loaded via Three.js GLTFLoader with interactive drag rotation
- GSAP ScrollTrigger animations with pinned sections and scrub
- Lenis smooth scrolling
- Scroll-driven donut color changes per flavor section
- Card stack reveal animation (Pistachio, Classic Glaze, Berry Pop, Choco Bliss)
- Numbered ordering steps with opacity transitions
- Mobile menu with slide-in animation
- Fully responsive (1600px down to 375px + height breakpoints)
